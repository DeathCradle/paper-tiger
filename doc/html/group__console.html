Detailed Description

The console and command subsystem uses libuv tty handles to read input from stdin (or another source) in a cross-platform manner, and pass the string input to the command system once the parameters have been parsed.

int console_init ( struct console *  console)

Initializes a newly-allocated console context.

Initializes the libuv tty handle contained in console->console_handle to accept input to and from stdin.

Parameters
[in]consoleA pointer to the console object allocated with a call to console_new
Returns
0 if the initialization was successful, < 0 otherwise. If this function cannot return successfully, paper tiger will be unable to process console input.
