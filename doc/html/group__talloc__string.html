char char char char char char* talloc_asprintf_append_buffer ( char *  s,
const char *  fmt,
  ... 
)

Append a formatted string to another string.

This is a more efficient version of talloc_asprintf_append(). It determines the length of the destination string by the size of the talloc context.

Use this very carefully as it produces a different result than talloc_asprintf_append() when a zero character is in the middle of the destination string.

1 char *str_a = talloc_strdup(NULL, "hello world");
2 char *str_b = talloc_strdup(NULL, "hello world");
3 str_a[5] = str_b[5] = '\0'
4 
5 char *app = talloc_asprintf_append(str_a, "%s", ", hello");
6 char *buf = talloc_strdup_append_buffer(str_b, "%s", ", hello");
7 
8 printf("%s\n", app); // hello, hello (app = "hello, hello")
9 printf("%s\n", buf); // hello (buf = "hello\0world, hello")

If s == NULL then new context is created.
