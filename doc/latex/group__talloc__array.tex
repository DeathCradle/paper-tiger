Provide a function version of talloc\+\_\+realloc\+\_\+size. 

This is a non-\/macro version of talloc\+\_\+realloc(), which is useful as libraries sometimes want a ralloc function pointer. A realloc() implementation encapsulates the functionality of malloc(), free() and realloc() in one call, which is why it is useful to be able to pass around a single function pointer.
